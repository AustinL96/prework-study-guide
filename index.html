HTML

  • The head element contains information about the webpage
  • The head element's information is not visible when a user looks at a webpage.
  • The body element represents the visible content shown to the user
  • The head element's information uses metadata to help the browser render the page correctly. This includes data like the character set, description, keywords, author, and viewports.
  • The "charset" element sets our character encoding.
  • The "http-equiv" element sets an HTTP header for content.
  • The "viewport" element sets the default view for our webpage.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to the main branch if your are not in it
  • pwd: displays the current working directory, and stands for "print working directory"
  • ls: displays a list of files and directories
  • cd "name": changes directory
  • cd ..: navigates your directory backwards
  • mkdir "name": makes a new directory
  • touch "name": creates a new file inside a directory
  • start "name": starts a file
